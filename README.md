## 📌 မိတ်ဆက်

**MASTER 2D AMO** သည် မြန်မာနိုင်ငံတွင်း 2D ထီရောင်းချသူများအတွက် ရည်ရွယ်ဖန်တီးထားသော **အချိန်နှင့်တပြေးညီ စာရင်းစစ်စနစ်** ဖြစ်ပါသည်။

ဤစနစ်တွင် အောက်ပါ လောဂျစ်မျိုးစုံကို ထောက်ပံ့ထားသည် -
- ပါဝါ၊ နတ်က္ခ၊ အပူး၊ ညီကို
- ထိပ်၊ ဘိတ်၊ ပွင့်
- ခွေ၊ အပူးပါခွေ
- အပါ၊ အပူး+အပါ
- **R** (ရှော့/လှန်) စနစ် (အပူးဂဏန်းများတွင် R အလုပ်မလုပ်ပါ)

## 🎯 အင်္ဂါရပ်များ

| အင်္ဂါရပ် | အသေးစိတ် |
|-----------|-----------|
| ✅ **အရောင်းစနစ် (POS)** | လောဂျစ်အားလုံးပါဝင်ပြီး ဂဏန်းများကို ခလုပ်များဖြင့် အလွယ်တကူ ရိုက်ထည့်နိုင် |
| ✅ **Sniper Live** | Thai Stock 2D API မှ တိုက်ရိုက်ဒေတာဆွဲယူပြီး ညနေ ၄ နာရီတွင် သော့ခတ်ကာ 2D ခန့်မှန်းချက်ထုတ်ပေး |
| ✅ **လယ်ဂျာ (Ledger)** | ဂဏန်း ၀၀-၉၉ အထိ စာရင်းအပြည့်၊ တစ်ကွက်ချင်းစီ ပြင်ဆင်နိုင် |
| ✅ **QR Code** | လယ်ဂျာကို QR အဖြစ်ထုတ်ယူခြင်း၊ အခြား device မှ ဖတ်သွင်းခြင်း |
| ✅ **History** | ဘောင်ချာမှတ်တမ်းများကို ပြင်ဆင်ခြင်း၊ ဖျက်ခြင်း ပြုလုပ်နိုင် |
| ✅ **ဘောင်ချာ (Voucher)** | PNG ပုံအဖြစ် ပရင့်ထုတ်နိုင်၊ အစမ်းကြည့်ရာတွင် စာရင်းများကို ပြင်ဆင်နိုင် |

## ⚙️ တပ်ဆင်အသုံးပြုနည်း

1. ဖိုင်အားလုံးကို ဒေါင်းလုဒ်ဆွဲပါ (သို့) clone လုပ်ပါ။
2. `index.html` ကို browser ဖြင့်ဖွင့်ပါ။ (အင်တာနက်ချိတ်ဆက်ထားရန် လိုအပ်သည် - API အတွက်)
3. အသုံးပြုလိုသော လောဂျစ်ကို ခလုပ်များဖြင့် ရိုက်ထည့်ကာ `ENTER` နှိပ်ပါ။
4. ဘောင်ချာထုတ်ရန် "စမ်းထုတ်" ခလုပ်ကို နှိပ်ပါ။
5. **Sniper** tab တွင် တိုက်ရိုက် 2D ခန့်မှန်းချက်ကို ကြည့်ရှုနိုင်သည်။
6. **လယ်ဂျာ** tab တွင် စာရင်းများကို စစ်ဆေးနိုင်သည်။
7. **History** tab တွင် မှတ်တမ်းများကို ပြင်ဆင်/ဖျက်နိုင်သည်။
